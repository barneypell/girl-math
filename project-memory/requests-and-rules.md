# Girl Math Requests and Product Rules

## Confirmed Product Rules
- VIP pass is monthly.
- Standard price is $8.
- Friday price is $5 with a $3 Friday discount.
- Snack bar items are included free with VIP.
- Only non-consumable spa items are included with VIP.
- Consumable spa services such as anything requiring cream or shampoo become separate paid add-ons.
- Worker view must clearly show what is included versus what costs extra.

## Confirmed UX / Workflow Rules
- Admin tools must be on a separate page so they do not distract customers.
- The main customer page must include a visible link at the top to the admin tools page.
- Users must not get VIP automatically.
- VIP requires explicit admin approval.
- Admin view must support approve and reject actions for VIP requests.

## Confirmed Creative Corrections
- Spa Treatments and Weekly Comics are separate sections.
- Comics Corner must not appear as a spa treatment menu option.
- Name tag selections should render as actual graphic items on the designed tag, not text labels like "heart" or "gems".
- Comic scenes should use Alexandra's actual drawing layouts as the source structure.
- Comic scenes may be adapted to anime style only.
- Do not change the comic words or their positioning.

## Operational Requirements
- Keep a persistent store for project instructions, artifacts, history, requests, status, and plans.
- Make the structure durable across restarts and context compression.
- Support multiple concurrent projects without mixing context.
- Support easy switching between projects and easy querying across all projects.
