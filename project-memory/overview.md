# Girl Math Overview

## Project
Girl Math is a Streamlit-based app for a playful VIP pass / salon experience.

## Current Product Shape
- Shared VIP passes with pass creation, reopen by code, and email recovery.
- Request history and event logging.
- Separate customer page and worker/admin page.
- VIP request and admin approval workflow.

## Visual / Creative Source of Truth
- Alexandra's original art is the source of truth for the next design pass.
- The app should feel like Alexandra's original work, not a generic remake.

## Working Principles
- Preserve the playful handmade tone.
- Keep customer-facing screens clean.
- Keep worker/admin tools separate from the customer experience.
- Treat Barney's product corrections as hard requirements, not optional polish.
